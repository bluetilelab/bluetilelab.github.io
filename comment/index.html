    Comment

    C++에서 주석 (Comment) 은 개발자와 코드를 작업하거나 검토하는 다른 사람의 가독성과 이해도를 높이기 위해 소스 코드 내에 설명, 메모 또는 주석을 삽입하는 작업을 말합니다. 주석은 컴파일 과정에서 컴파일러가 무시하는 텍스트 줄로, 프로그램의 기능이나 실행에 영향을 미치지 않습니다. 주석의 유일한 목적은 사람이 읽을 수 있는 설명을 제공하는 것입니다.

    General

    Purpose

    일반적인 주석은 다음과 같은 목적으로 사용할 수 있습니다.

    • 변수, 함수, 구조체, 클래스 등에 대한 부가 설명
    • 함수 호출 목적 및 예상 동작 결과 설명

    How to Write

    일반적인 주석은 단일 라인 그리고 슬래쉬 2개 // 을 원칙으로 합니다.

    • 블록 주석 /* */은 일반적인 주석 이 아닙니다. 사용하지 않습니다.
    • 코드를 일괄 주석 처리하는 경우, 모든 라인을 // 로 주석처리!

    Language

    언어는 무관합니다. 다만 저희는 한국사람이므로 영문보다는 한글을 선호합니다.

    일반 주석
    QTableWidget* tw; // 검사 결과 테이블 (1)

// 로그 메시지를 로거로부터 얻음 (2)
auto message = logger->GetMessage();
    1. 변수 설명
    2. 동작 설명

    Note

    Purpose

    NOTE 주석은 다음과 같은 목적으로 사용할 수 있습니다.

    • 주석이 가리키는 부분에 대한 주의사항 을 명시
      • 반드시 00 가 초기화되어 있어야 한다.
      • 00 기능에서만 동작하는 코드
      • 00, XX 등 고객사의 요청사항
      • 동작 시점은 00 이다.
    • 비록 일반 주석이나 필요에 의해 공동 작업자가 알 수 있도록 강조하고 싶을 때

    How to Write

    NOTE 주석은 다음과 같은 포맷으로 작성합니다.

    • //: 주석 시작
    • NOTE: 식별자
    • @id: 작성자
    • date: 날짜
    • message: 메시지
    NOTE
    // NOTE @kanghee 2023510 매우 중요한 설정 (1)
QThreadPool::globalInstance()->setMaxThreadCount(thread_pool_size);

inspection_data->index = next++; // NOTE @kanghee 20230515 검사 결과 인덱스 (2)
    1. 공동 작업자에게 해당 부분이 중요하다는 사실을 인지하게 함
    2. 일반 주석을 강조
